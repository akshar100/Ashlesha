Ashlesha Framework

Ashlesha combines CodeIgniter, Twitter Bootstrap and YUI3 app framework to give you a responsive and next generation web application framework. This is an in-page app that provides progressively enhanced UI which adapts well tablets, mobile and desktop screens.

Many sites are built using Ashlesha for example "BrandABrand.com":http://www.brandabrand.com/ "E-Yantra Competition":http://portal.e-yantra.org/

And many more

Built for modern browsers and modern web

With the rise of HTML5 many things about the web are changing. Ashlesha keeps that in mind and integrates Facebook, Google Maps, Google Charts and Dropbox with your app. It's admin console is rich and provides configurable analytic.

Built for Performance and Scalability

Ashlesha follows Javascript performance guidelines very strictly. It's code is one of the finest Javascript code. (And if you disagree you are welcome to send pull requests). It heavily uses client side caching and local storage to avoid unnecessary server calls. Database support is provided through CouchDB. It is also integrated with lucene for full text search.

Built for intelligent behavior

ARDRA is a Python framework for processing the data generated by Ashlesha based web frameworks.
